Clicktionary

Clicktionary is a Chrome extension that lets you click on any Spanish word while browsing and save it to a personal word bank. Words can then be studied Quizlet-style with flashcards directly in the extension. Coming to the Chrome Store soon!

Through this project I've learned and practiced:

  • Chrome Extension architecture and APIs
  • Backend integration with Supabase
  • Product management and scoping for browser tools
  • HTML/CSS for extension UI
