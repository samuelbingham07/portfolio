Dance Game

A Just Dance-inspired movement game built in p5.js for a class project with a group. It uses your webcam and PoseNet to track your body in real time, scoring you based on how well you match the dance moves shown on screen. Inspired by Just Dance and Tomodachi Life.

How It Works

PoseNet continuously reads keypoints from the webcam feed — joints like elbows, shoulders, wrists, and hips. Rather than comparing full pose similarity scores, the scoring system checks whether specific geometric conditions are met for each move: things like whether an elbow is above a shoulder, or whether a wrist has crossed the midline of the body. Each dance move is defined as a set of these conditions, and you score points when your body satisfies them within the timing window.

The Hard Part

The scoring system was entirely manual — every condition for every pose had to be hand-authored by figuring out which joint relationships actually captured what the move looked like, then tuning the thresholds so the game felt fair without being too strict or too forgiving. There's no ground truth to work from: you're translating a visual dance move into a set of boolean geometric rules, and getting that translation right for every move in the game took most of the development time.

Play it yourself

  • p5.js for interactive, browser-based creative coding
  • Translating visual movement into programmatic geometric conditions
  • Manual scoring system design — defining, tuning, and balancing pose detection thresholds
  • Collaborating on a creative coding project with a group under a class deadline
