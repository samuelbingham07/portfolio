Side Projects

Here are some of the smaller projects I've worked on in my free time!

Narrative Game

This is a narrative game I made using p5.js! It features a branching storyline where your choices affect the outcome.

Explore for yourself!

Dance Game

This is a dance game I made using p5.js and PoseNet! It uses your webcam to track your movements and score you based on how well you mimic the dance moves shown on screen. Inspired by Just Dance and Tomodahi Life.

Get some exercise and play!

TheVault.

This was a fun project I did for a designathon that protects elders against impersonation scams by creating a familiar, digital phonebook that helps them verify the identity of their contacts if they are suspicious of scam callers.

Check out the demo I made using Lovable!

Check out our final presentation!

Machine Learning Art

This project was super random but I used ML to generate art! Not sure how I feel about AI art but I feel like I just used it as a tool here but I wouldn't dare to call it real art! Along with the video above, I made a bunch more that got lost in my Machine Learning class. I learned AnimateDiff, DeForum, Lora models, Control Net, Deep Dream, Stable Diffusion, IPAdapter, and t-SNE to create this and the others.
