Mumei

Mumei is my first game built with Unity! I collaborated with my friend to develop a Japanese-learning RPG from scratch.

Github Repository

Through this project, I’ve learned about and practiced the game development process, including:

  • Game design principles
  • Unity engine capabilities, usage, and C#
  • How all major game engines work (Unreal, Godot, GameMaker, Lua)
  • Storytelling and narrative/character design
  • 2D/3D art (Aseprite, Procreate, Blender)
