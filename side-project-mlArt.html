Machine Learning Art

This project came out of a course called ML for Artists — a class that treated machine learning as a creative medium rather than an engineering tool. The concept I explored was video game addiction: the psychological phenomenon where someone becomes so deeply invested in a game that the boundary between themselves and their character begins to dissolve. They don't just play the character — they start to see themselves as the character.

The video above is the surviving piece from a larger series. I used a stack of ML tools — AnimateDiff, DeForum, Stable Diffusion, LoRA models, ControlNet, IPAdapter, and Deep Dream — to produce generated video that visualizes that identity collapse: a human figure slowly morphing into and becoming indistinguishable from a game character. The other pieces from the series were unfortunately lost. I have mixed feelings about AI-generated art as a category, but using these tools to say something specific about a real psychological experience felt different from using them as a shortcut — more like using a medium to make a point about the medium itself.

Through this project I've learned and practiced:

  • Stable Diffusion and LoRA model fine-tuning for consistent character and style generation
  • AnimateDiff and DeForum for AI video generation and frame interpolation
  • ControlNet and IPAdapter for guided image generation with reference inputs
  • Deep Dream for psychedelic, recursive visual transformation
  • t-SNE for visualizing high-dimensional data embeddings
  • Using ML tools with a deliberate creative concept rather than generating for its own sake
